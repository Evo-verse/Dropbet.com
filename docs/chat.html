Chat Toggle Logic

The chat is opened and closed by switching the value in the store useSidebarStore:

const stripOpen = useSidebarStore((store) => store.stripOpen);
const toggleStrip = useSidebarStore((store) => store.toggleStrip);

Currently, the mechanism for hiding the chat is not implemented, and to hide the chat completely, you need to comment out the following code:
