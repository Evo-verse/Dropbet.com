apps/api/src/bet/bet.service.ts

Description

READ ME This service allows flexible creating/settling of bets. Each user's gambling action should be processed only by BetService service.

Important:

  • Each bet has BetIdentity that can be created by bet-helper.ts. It uses for generating unique ids. It gives:
    • The same bet.id even if there is no GameId.
    • Impossible to create more than ONE bet cause BetIdentity is generated by seed data.
  • Bets are used for complete gambling statistics.

Details:

  • Bet.amount and Bet.payout describes a user's balance AND loss item's cost.
  • Settled bet cannot be changed
