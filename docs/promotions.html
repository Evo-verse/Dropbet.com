Promotions & Blogs

Promotions & Blogs Management

Promotions and blogs are currently hard-coded in src/constants/promotions.tsx and src/constants/blog.tsx. There is no way to manage promotions/blogs from the admin panel at this time.

Adding a New Promotion/Blog

To add a new promotion/blog, you need to add a new item to the promotionsPosts/blogPosts list and update the EPromotionTitle/EBlogTitle enum. 

export enum EPromotionTitle {
    DEPOSIT_BONUS = 'Welcome Bonus',
}

export type TPromotionsPost = (typeof promotionsPosts)[0];

export const promotionsPosts = [
    {
        title: EPromotionTitle.DEPOSIT_BONUS,
        get href() {
            return sanitizePromoUrl(this.title);
        },
        img: depositBonusImg,
        date: new Date(new Date().setDate(new Date().getDate() + 30)).toString(),
        description: `Everything You Need to Know About Dropbet’s 100% Welcome Bonus`,
        content: <Promotion />,
    },
];


export enum EBlogTitle {
  $5K_Raffle = '$5k Raffle',
  WAGER_$25k = 'Wager $25k get $100',
  BECOME_AFFILIATE = 'Become an affiliate at Dropbet',
  LOSSBACK = 'Lossback for the first 7 days',
}

export type TBlogPost = (typeof blogPosts)[0];

export const blogPosts = [
  {
    title: EBlogTitle.LOSSBACK,
    get href() {
      return sanitizeBlogUrl(this.title);
    },

 ...
];

How It Works

This list is displayed on the promotions/blogs list page:

src/components/pages/promotions/index.tsx;

src/components/pages/blog/index.tsx;

We pass a link to the promotion/blog list element that matches the promotion/blog title. This allows navigation to a specific promotion/blog page to view details.

The content for each promotion/blog is provided as a React component. For example: 

content: <Promotion />, - promotion

or

content: <Lossback />, - blog
