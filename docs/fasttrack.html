FastTrack Integration

Scope of Data Transferred

The integration with FastTrack receives the following data points from our system:

  • Bet Activity: Bet amounts, metadata, and game identifiers.
  • Game Participation: List of games where bets are placed.
  • Deposits: User deposit transactions and amounts.
  • Promotional Progress: Progress markers in active promotional programs.
  • Withdrawals: User withdrawal requests and completed transactions.

Usage

  • Analytics: FastTrack processes the data to provide more accurate activity and behavioral analytics.
  • Personalization: Powers the frontend recommendation engine, enabling personalized offers and campaigns.
  • Backend Promotion Handling: Integration allows FastTrack to credit users with additional funds as part of promotional campaigns.

Promo System

System Representation

Exposed under the Promo module.

Users are tracked via the Promo Tracker once they engage with a promotional program.

Behavior

  • Registration: When a user opts in, they are registered in the tracker.
  • Deposit Trigger: Upon the user’s first deposit, if a deposit promo is active, funds are credited automatically according to the configured promo rules.
