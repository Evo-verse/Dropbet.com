15. Home Page Grid (Not Working ATM)

Purpose

The Home Page Grid section allows you to control which games appear on the casino’s main page and how they are organized. You can choose game categories, select specific games, and manage their display order for players.

Categories

Games are grouped into main categories:

  • House Games – In-house developed games such as Speed Roulette, Blackjack, Limbo, Mines, and Plinko.
  • Live Casino – Real-time dealer games from providers like Evolution, Ezugi, and Pragmatic Play.
  • Slots – Video slots from various providers.

Editing a Category

  1. Go to Home Page Grid in the admin panel.
  2. Locate the category you want to edit (House Games, Live Casino, or Slots).
  3. Click Edit to open the category configuration.

Managing Games

Inside a category’s Edit view:

  • Add Games – Select from the list of available enabled games on the platform.
  • Remove Games – Click the delete icon to remove a game from the category’s display.
  • Reorder Games – Drag and drop games to change the order in which they appear on the home page.
  • Preview – See how the arrangement will look before saving.

Saving Changes

After making adjustments, click Save to apply them immediately to the live casino home page.
