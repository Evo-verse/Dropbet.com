3. Users Management

The Users Management module provides a full overview of all registered players, their activity, and account details. It allows admins to filter, search, and review player statistics in real time.

3.1 Key Stats Overview

At the top of the page, four live counters summarize platform user activity:

  • New Users – Number of newly registered accounts in the selected period.
  • Online – Current count of users actively online.
  • Total Users – Total registered accounts.
  • Restricted Users – Accounts with restricted access due to bans, compliance issues, or manual suspension.

3.2 Filters

Admins can refine user lists using the following filters:

  • VIP Level – Filter by minimum and maximum VIP tier.
  • Total Wagered – Filter by total bet amount range.
  • LTV – Lifetime value range.
  • Balance – Current wallet balance range.
  • Registration Date – Filter by specific sign-up period.

The Hide Filters button collapses the filter section for more workspace. Clear resets all filters.

3.3 Search

The search bar allows quick lookup by username or email address. Clicking Search applies the term alongside any active filters.

3.4 User List Table

The table displays all users matching the filter/search criteria with the following columns:

  • User – Player’s username with profile icon.
  • Email Address – Registered email.
  • VIP Level – Current loyalty tier.
  • Total Wagered – Cumulative bets placed.
  • LTV – Lifetime value of the player.
  • Last Active – Time since last login.
  • Referral Code – Code used during registration (if any).
  • Balance – Current wallet balance.
  • Register Date – Exact date and time of account creation.
  • More – Action menu for user-specific management options (e.g., view full profile, adjust balance, apply restrictions).
