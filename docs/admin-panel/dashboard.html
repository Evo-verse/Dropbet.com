2. Dashboard

Purpose

The Dashboard provides detailed operational analytics for the casino platform, broken down into multiple tabs: General, Games, Game by Slug Stats, and Finances. This section is used to monitor platform performance, user activity, and financial trends in real time.

2.1 General Tab

Displays high-level KPIs and trends over the selected time period.

Metrics & Graphs:
  • Reg Count / FTD % – Total number of registered users vs. first-time depositors, shown as counts and conversion percentage.
  • Deposit Amount / In/Out Ratio – Compares deposit totals, withdrawal totals, and the in/out ratio percentage.
  • Active Players – Number of unique active players within the selected time frame.
  • Turnover / ATPU – Total bet turnover alongside Average Turnover Per User.
  • GGR / ARPU – Gross Gaming Revenue compared to Average Revenue Per User.
  • NGR All – Net Gaming Revenue across the platform.
Controls:
  • Date range selector (e.g., Today, Last 24 hours)
  • Time granularity selector (By Hour, By Day)
  • Visibility toggles for each graph

2.2 Games Tab

Focuses on performance metrics for all games combined.

Metrics & Graphs:
  • Active Players – Total active players per time slot.
  • Turnover / ATPU – Total wagers and average turnover per user.
  • GGR / ARPU – Gross Gaming Revenue and average revenue per user.
  • Game Type by Margin – Bar chart comparing margin percentages for different game categories (e.g., Slots, House Games).
  • Game Type by Turnover – Pie chart showing turnover distribution between game categories.
Filters:
  • Game selection dropdown
  • Date range selector
  • Time granularity selector

2.3 Game by Slug Stats Tab

Provides per-game breakdowns.

Table Columns:
  • Game Slug & Name – Internal identifier and game title.
  • Category – Game category (e.g., Slots, House Game).
  • Provider – Name of the game provider.
  • Turnover – Total wagered amount for the game.
  • GGR – Gross Gaming Revenue for that game.
  • Average Revenue – Average revenue generated per session.
  • Average Turnover – Average wager amount.
  • Bet Count – Number of bets placed.
  • Active Players – Unique players who played the game.
  • Margin – Profitability percentage for the game.
  • More Info – Link or menu for detailed data.
Filters:
  • Game name
  • Game slug
  • Category
  • Provider
  • Date range

2.4 Finances Tab

Focuses on financial transaction data.

Metrics:
  • Wallet Change – Net change in user balances.
  • Deposits – Total deposited amount.
  • Total Deposits – Number of deposit transactions.
  • Average Deposit – Mean deposit size.
  • Withdrawals – Total amount withdrawn.
Filters:
  • Include/Exclude test transactions
  • Date range picker
