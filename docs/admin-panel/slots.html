11. Slots

The Slots tab is where administrators can manage all slot games available on the platform, including enabling/disabling games, editing their details, and reviewing provider and RTP information.

Main Table Overview

Each row represents a slot game with the following columns:

  • ID – Unique system ID for the slot.
  • Icon – Displays the slot’s current thumbnail; if none is uploaded, a default placeholder is shown.
  • Default Icon – Indicates if the current icon is the default provider image.
  • Name – Full name of the slot game.
  • Provider – Name of the game provider and integration type (with a “Disabled” tag if the provider is inactive).
  • RTP – The game’s Return to Player percentage.
  • Category – The game category (e.g., Slots, Casual, Live Casino, etc.).
  • Active Toggle – Enables or disables the game for players. Disabled games remain in the system but cannot be played.
  • Actions – “Edit Slot” button to open the slot configuration window.
  • More Info – Three-dot menu for additional actions or quick details.

Editing a Slot

Clicking Edit Slot opens the slot management window with two tabs:

  1. Edit Slot Info
    • Image Upload – Allows replacing the slot’s thumbnail image by uploading a new file.
    • Category Selector – Assigns the game to one or more categories (House Game, Live Casino, Slots, Blackjack, Roulette, Game Shows, etc.).
    • Confirm / Cancel Buttons – Save or discard changes.
  2. Details Tab
    • Game Name – Official slot title.
    • Provider Name – The provider’s full name.
    • Provider Slug – Technical identifier for integration.
    • RTP – Displays the RTP value set by the provider or manually.
    • Game Info JSON – Shows technical metadata about the game, such as payout percentages or internal flags.
    • Thumbnail Preview – Displays the current slot image.
