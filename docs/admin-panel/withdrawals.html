8. Withdrawals

Withdrawals – Pending Review

This section of the admin panel displays all withdrawal requests that have been submitted by users but have not yet been processed.

Displayed Information

  • ID – Unique withdrawal request ID.
  • User – Username and avatar of the requester.
  • Amount – Requested withdrawal amount and currency (e.g., 10 USDT).
  • USD Amount – Value converted to USD.
  • Payment Provider Status – Status from the payment provider (e.g., Pending).
  • Transaction Hash – Blockchain transaction hash (empty until processed).
  • Transaction ID – Internal reference ID.
  • Date – Date and time of the withdrawal request.
  • Stats – User-related statistics, including:
    • VIP level
    • LTV (Lifetime Value)
    • Total wagered
  • Finance Stats – Player's deposit, withdrawn amount, and net difference.
  • Admin Action – Buttons for Approve or Reject the withdrawal.

Actions Available for Admins

  • Approve – Triggers a 2FA verification prompt. After confirmation, processes withdrawal via payment provider, updating transaction hash and status automatically.
  • Reject – Prompts for reject reason, stores it in record, notifies user, and moves request to Rejected tab.

Withdrawals – Approved

This section lists all withdrawal requests that have been processed and successfully completed through the payment provider.

Displayed Information

  • ID – Unique withdrawal request ID.
  • User – Username and avatar of the requester.
  • Amount – Withdrawn amount and currency (e.g., 50 USDC).
  • USD Amount – Value converted to USD.
  • Payment Provider Status – Always displays Complete for approved withdrawals.
  • Transaction Hash – Clickable blockchain transaction link for verification.
  • Transaction ID – Internal system reference for the withdrawal.
  • Date – Date and time when the withdrawal was processed.
  • Stats – User-related statistics:
    • VIP level
    • LTV (Lifetime Value)
    • Total wagered amount
  • Finance Stats – Player’s deposit total, withdrawn total, and net difference.
  • Admin username that approved the transaction.

Withdrawals – Rejected

This section lists all withdrawal requests reviewed and denied by an admin.

Displayed Information

  • ID – Unique withdrawal request ID.
  • User – Username and avatar of the requester.
  • Amount – Requested amount and currency (e.g., 50 USDC).
  • USD Amount – Value converted to USD.
  • Payment Provider Status – Denied.
  • Transaction Hash – Empty for rejected withdrawals.
  • Transaction ID – Internal reference number.
  • Date – Date and time of the withdrawal request.
  • Stats – User-related statistics:
    • VIP level
    • LTV (Lifetime Value)
    • Total wagered amount
  • Finance Stats – Player’s deposit total, withdrawn total, and net difference.
  • Admin Action – Shows admin username, avatar, and timestamp of rejection.
  • Status – Denied.
  • Reject Reason – Visible via a red info icon next to status, showing admin-provided reason on hover or click.

Withdrawals – Failed

This section lists all withdrawal requests unsuccessful due to payment provider issues rather than admin decisions.

Displayed Information

  • ID – Unique withdrawal request ID.
  • User – Username and avatar of the requester.
  • Amount – Requested amount and currency (e.g., 0.25 LTC).
  • USD Amount – Value converted to USD.
  • Payment Provider Status – Failed.
  • Transaction Hash – Usually empty (transaction not confirmed).
  • Transaction ID – Internal reference number.
  • Date – Date and time of the withdrawal request.
  • Stats – User-related statistics:
    • VIP level
    • LTV (Lifetime Value)
    • Admin Action – Displays admin details if manually retried or rejected after failure.
    • Status – Failed.

    Why Withdrawals Fail

    • Insufficient balance on the payment provider account.
    • Temporary blockchain network issues.
    • Invalid or unreachable withdrawal address.
    • Provider downtime or API errors.

    Next Steps for Failed Withdrawals

    • Retry – Admin can retry sending funds once issue is resolved.
    • Reject – If unresolved, admin can reject with a reason, moving request to Rejected tab.

  • Finance Stats – Player’s deposit total, withdrawn total, and net difference.
