Promocodes

Overview

The Promocodes page allows administrators to create, manage, and track promocodes that act as balance top-ups for players. Promocodes can be configured with specific amounts, usage limits, eligibility requirements, and expiration dates. They can be distributed for promotional campaigns, influencer giveaways, or loyalty rewards.

Page Layout & Features

1. Promocode List

The main table displays all existing promocodes with the following columns:

  • Promocode – Unique code players will enter to redeem the balance.
  • Title – Internal title for campaign identification.
  • Created – Date and time the promocode was created.
  • Amount ($) – Value of the promocode in the set currency.
  • Claims Left – Remaining number of times the code can be redeemed (blank if unlimited).
  • Status – Shows whether the code is Active or Inactive.
  • Total Users Claimed – Number of unique users who have redeemed it.
  • Expires – Date and time when the code expires.
  • Actions – Three-dot menu to edit or delete the promocode.

Use the Search bar to find specific codes or the Choose Filter dropdown to refine results.

2. Adding a New Promocode

Click + Add Promocode to open the creation form. Required fields are marked with *.

  • Promocode* – Unique code string (case-insensitive).
  • Title* – Internal campaign name.
  • Amount ($)* – Credit amount given upon redemption.
  • Currency* – Currency type (e.g., DBC).
  • User KYC Level – Minimum verification level required to redeem.
  • Affiliate Code – Optional affiliate link field.
  • Min VIP Level – Minimum VIP tier required.
  • Min Total Wager ($) – Minimum wagering requirement.
  • Min Total Deposit ($) – Minimum deposit requirement.
  • Total Claims – Maximum total redemptions across all players.
  • Lock Withdrawal Hours – Hours withdrawals are locked after redemption.
  • Expiration Date* – Date/time after which the code is invalid.
  • Hidden – Hide from public listings.
  • Active – Activate immediately upon creation.
  • Create Batch – Generate multiple codes with same configuration.

Actions:
• Submit – Save the promocode.
• Cancel – Discard changes.

3. Editing or Deleting Promocodes

Use the Actions menu to modify or remove an existing promocode. Changes to active codes take effect immediately.
