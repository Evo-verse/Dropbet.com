12. Leaderboards

The Leaderboards section allows you to manage and schedule competitive events for users, rewarding top performers with prizes.

Tabs Overview:

  • Active – Shows currently running leaderboards.
  • Settlement – Displays leaderboards that are in the process of finalizing results and distributing rewards.
  • Finished – Contains completed leaderboards with historical data.
  • Scheduler – Used to configure and schedule upcoming leaderboards (Daily, Weekly, Monthly).

Scheduler Functions:

  • Type – Select the leaderboard frequency: Daily, Weekly, Monthly.
  • Ends in – Countdown timer for the leaderboard's current cycle.
  • Prize pool – Displays the total prize amount for that leaderboard cycle.
  • Enabled – Toggle to activate or deactivate the leaderboard.
  • Config next – Opens the configuration panel for the next cycle.

Configuring the Next Leaderboard:

  1. Name – Title of the leaderboard event.
  2. Enable – Toggle to activate it.
  3. Description – Optional text describing the event.
  4. Prizes – Define prize amounts for each position (e.g., Prize 1: 30,000, Prize 2: 10,000, Prize 3: 5,000). Additional prizes can be added via Add Prize.

Click Configure next leaderboard to save changes.
