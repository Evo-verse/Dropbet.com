1. Overview

Purpose

The Overview page provides administrators with a real-time snapshot of the platform’s performance and user activity. It consolidates key financial indicators, operational statistics, and player engagement metrics into a single dashboard.

Top Filters

  • Select Categories to Show – Filters the KPIs displayed in the grid.
  • Date Range Selector – Choose predefined ranges (e.g., Last 24 hours) or custom start/end dates.
  • Main Statistics Grid – A collection of metric tiles showing:
    • Current value
    • Percentage change vs. previous equivalent period
    • Tooltip with metric definition

Metrics Displayed

  • NGR (Net Gaming Revenue) – Revenue after subtracting wins, bonuses, and rakeback.
  • GGR (Gross Gaming Revenue) – Total wagers minus wins before deductions.
  • Rakeback – Amount paid out to players as rakeback.
  • Affiliate Earnings – Total affiliate commissions paid.
  • Leaderboard – Total prizes distributed via leaderboards.
  • Admin Tips – Total tips sent from admins to players.
  • Profit/Loss – Net operational result for the selected period.
  • Deposits – Total deposit amount from all players.
  • Average Deposit Amount – Mean deposit size.
  • Withdrawals – Total withdrawal amount.
  • FTDs (First-Time Deposits) – Count of users making their first deposit.
  • FTDs Average – Average value of first deposits.
  • Online – Number of currently active users.
  • Users Balance on Site – Total funds currently held in all player accounts.
  • Total Users – Count of all registered accounts.
  • New Registrations – Number of new player sign-ups.
  • Wagered – Total value of bets placed in the selected period.
  • User Tips – Tips sent between players.

Usage

  • Select the desired date range.
  • Filter displayed KPIs via Select Categories to Show.
  • Hover over info icons to view metric explanations.
  • Monitor percentage change indicators for anomalies or trends.
